# Git

### معرفی Git

Git یک سیستم کنترل نسخه توزیع شده رایگان و متن باز است که برای مدیریت همه کارها از پروژه های کوچک تا بسیار بزرگ با  کارایی بالا طراحی شده است.



### دستورات رایج گیت

- git init
- git add -A
- git commit
- git branch
- git checkout [branch]
- 'git push origin master
- git pull origin master
